# Comparative-F1-telemetry-analysis

## Overview
This project focuses on analyzing Formula 1 telemetry data using Python, MySQL, and Power BI. The project extracts real-world telemetry data of Formula 1 drivers and performs comparative analysis based on parameters such as speed, throttle, brake usage, and lap distance.

The project demonstrates an end-to-end data analytics workflow including data extraction, preprocessing, database integration, and dashboard visualization.

---

## Features
- Formula 1 telemetry data extraction using FastF1
- Data preprocessing using Pandas
- MySQL database integration
- Comparative driver analysis
- Interactive Power BI dashboard
- Speed vs Distance analysis
- Throttle vs Distance analysis

## Dataset
Telemetry data was extracted from the Formula 1 2024 Suzuka Qualifying Session using the FastF1 library.

Drivers analyzed:
- Max Verstappen (VER)
- Charles Leclerc (LEC)
